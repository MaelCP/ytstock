# ytstock Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Maintenir en permanence ~15 Go de vidéos YouTube non vues (abonnements + recommandations + thèmes), qui se suppriment quand on les regarde (local via `lsof`, online via l'historique) et se recomplètent automatiquement en arrière-plan.

**Architecture:** Un seul script Python 3 stdlib (`ytstock.py`) qui orchestre les binaires `yt-dlp` et `aria2c`. Sous-commandes `daemon` / `refill` / `status`. État en fichiers plats (`.ytstock/seen.txt`, `.ytstock/watch.json`). Lancé au boot par un LaunchAgent. La logique pure (extraction d'ID, filtres, budget, transitions du watcher) est isolée et testée par `assert`; les parties I/O réelles (yt-dlp, lsof, launchd) sont validées manuellement.

**Tech Stack:** Python 3 (stdlib uniquement), yt-dlp 2026.07.04, aria2c 1.37.0, launchd (macOS).

## Global Constraints

- **Stdlib Python uniquement** — aucune dépendance pip. Seuls binaires externes : `yt-dlp`, `aria2c`.
- **Budget disque : 15 Go** (`BUDGET_BYTES = 15 * 1024**3`).
- **Qualité : 720p max** (`-f "bv*[height<=720]+ba/b[height<=720]"`).
- **Cookies : `--cookies-from-browser chrome`** sur tout appel yt-dlp qui touche le compte.
- **Téléchargeur : `--external-downloader aria2c`**.
- **Dossier cible : `/Users/satoshi/Downloads/videos`** ; nommage yt-dlp `%(title)s [%(id)s].%(ext)s`.
- **Thèmes : philosophie, ARTE, psychologie, documentaire, NBA.**
- **Le démon ne meurt jamais sur une erreur de source** : chaque étape isolée en try/except → log + continue.
- **Tests : `assert` sans framework**, exécutables via `python3 ytstock.py --self-check`.
- ID YouTube = 11 caractères `[A-Za-z0-9_-]`.

def is_wanted(cand, seen):
    if cand["id"] in seen:
        return False
    if cand.get("live_status") in ("is_live", "is_upcoming"):
        return False
    d = cand.get("duration")
    if d is not None:
        if d < MIN_DURATION:
            return False
        if MAX_DURATION and d > MAX_DURATION:
            return False
    return True

```python
import json
import os


def ensure_state_dir():
    STATE_DIR.mkdir(parents=True, exist_ok=True)


def _atomic_write(path, text):
    tmp = path.with_suffix(path.suffix + ".tmp")
    tmp.write_text(text)
    os.replace(tmp, path)


def load_seen():
    if not SEEN_FILE.exists():
        return set()
    return {ln.strip() for ln in SEEN_FILE.read_text().splitlines() if ln.strip()}


def add_seen(video_id):
    seen = load_seen()
    if video_id in seen:
        return
    with open(SEEN_FILE, "a") as f:
        f.write(video_id + "\n")


def load_watch():
    if not WATCH_FILE.exists():
        return {}
    try:
        return json.loads(WATCH_FILE.read_text())
    except (ValueError, OSError):
        return {}


def save_watch(d):
    _atomic_write(WATCH_FILE, json.dumps(d))
```

### Task 5: Logique de transition du watcher (pure)

**Files:**
- Modify: `ytstock.py`

**Interfaces:**
- Consumes: `WATCHED_SECS`, `POLL_SECS`.
- Produces: `watcher_tick(open_ids: set, acc: dict) -> tuple[set, dict]` — renvoie `(watched_ids, new_acc)`. Règle : pour tout id ouvert, `new_acc[id] = acc.get(id,0)+POLL_SECS`. Pour tout id présent dans `acc` mais plus ouvert : s'il avait `>= WATCHED_SECS` → dans `watched_ids` ; dans tous les cas il quitte `new_acc` (compteur remis à zéro en le retirant).

- [ ] **Step 1: Écrire le test**

```python
    # ouverture progressive
    watched, acc = watcher_tick({"a"}, {})
    assert watched == set() and acc == {"a": POLL_SECS}
    # continue jusqu'au seuil (POLL_SECS=15 → 6 tics = 90s)
    for _ in range(5):
        watched, acc = watcher_tick({"a"}, acc)
    assert acc["a"] >= WATCHED_SECS and watched == set()
    # fermeture après seuil atteint => vue
    watched, acc = watcher_tick(set(), acc)
    assert watched == {"a"} and "a" not in acc
    # fermeture avant seuil => pas vue, oublié
    _, acc2 = watcher_tick({"b"}, {})           # b à 15s seulement
    watched, acc2 = watcher_tick(set(), acc2)
    assert watched == set() and "b" not in acc2
```

- [ ] **Step 3: Implémenter**

```python
def watcher_tick(open_ids, acc):
    new_acc = {}
    watched = set()
    for vid in open_ids:
        new_acc[vid] = acc.get(vid, 0) + POLL_SECS
    for vid, secs in acc.items():
        if vid not in open_ids and secs >= WATCHED_SECS:
            watched.add(vid)
    return watched, new_acc
```

import datetime


def log(msg):
    line = f"{datetime.datetime.now().isoformat(timespec='seconds')} {msg}"
    print(line)
    try:
        ensure_state_dir()
        with open(LOG_FILE, "a") as f:
            f.write(line + "\n")
    except OSError:
        pass


def path_for_id(video_id):
    for p in list_stock_files():
        if id_from_name(p.name) == video_id:
            return p
    return None


def mark_watched(video_id, reason):
    add_seen(video_id)
    p = path_for_id(video_id)
    if p and p.exists():
        p.unlink()
        log(f"watched ({reason}) -> deleted {p.name}")
    else:
        log(f"watched ({reason}) {video_id} (no local file)")

### Task 7: Lister les candidats via yt-dlp (I/O réel)

**Files:**
- Modify: `ytstock.py`

**Interfaces:**
- Produces:
  - `list_source(source: str, limit: int) -> list[dict]` — appelle yt-dlp en `--flat-playlist` et renvoie des `{"id","duration","live_status"}`.
  - `gather_candidates(limit_per_source: int) -> list[dict]` — concatène, dans l'ordre de priorité, `:ytsubs`, `:ytrec`, puis `ytsearch{limit}:"<thème>"` pour chaque thème. Dédoublonne par id en gardant la 1re occurrence.

- [ ] **Step 1: Implémenter (glue I/O — testé manuellement)**

```python
import subprocess


def list_source(source, limit):
    cmd = [
        "yt-dlp", "--flat-playlist", "--ignore-errors",
        "--cookies-from-browser", "chrome",
        "--playlist-end", str(limit),
        "--print", "%(id)s\t%(duration)s\t%(live_status)s",
        source,
    ]
    try:
        out = subprocess.run(cmd, capture_output=True, text=True, timeout=300)
    except (subprocess.SubprocessError, OSError) as e:
        log(f"list_source error {source}: {e}")
        return []
    cands = []
    for line in out.stdout.splitlines():
        parts = line.split("\t")
        if len(parts) != 3 or len(parts[0]) != 11:
            continue
        vid, dur, live = parts
        duration = int(dur) if dur.isdigit() else None
        cands.append({"id": vid, "duration": duration,
                      "live_status": None if live in ("NA", "None", "") else live})
    return cands


def gather_candidates(limit_per_source):
    sources = [":ytsubs", ":ytrec"]
    sources += [f'ytsearch{limit_per_source}:"{t}"' for t in THEMES]
    seen_ids, out = set(), []
    for src in sources:
        for c in list_source(src, limit_per_source):
            if c["id"] not in seen_ids:
                seen_ids.add(c["id"])
                out.append(c)
    return out
```

### Task 9: Orchestration `refill()`

**Files:**
- Modify: `ytstock.py`

**Interfaces:**
- Consumes: `dir_used_bytes`, `needs_more`, `gather_candidates`, `is_wanted`, `download`, `add_seen`, `load_seen`.
- Produces: `refill()` — remplit jusqu'au budget. Idempotent.

- [ ] **Step 1: Implémenter**

```python
def refill():
    ensure_state_dir()
    used = dir_used_bytes()
    if not needs_more(used):
        log(f"refill: full ({used // 1024**2} MiB), rien à faire")
        return
    log(f"refill: {used // 1024**2} MiB utilisés, budget {BUDGET_BYTES // 1024**2} MiB")
    seen = load_seen()
    # limite par source large : on filtre beaucoup, on veut assez de candidats
    candidates = [c for c in gather_candidates(40) if is_wanted(c, seen)]
    log(f"refill: {len(candidates)} candidats retenus")
    for c in candidates:
        if not needs_more(dir_used_bytes()):
            break
        vid = c["id"]
        if download(vid):
            add_seen(vid)
        else:
            add_seen(vid)  # évite de re-tenter en boucle un ID cassé
    log(f"refill: terminé, {dir_used_bytes() // 1024**2} MiB")
```

### Task 11: `lsof` → fichiers ouverts + boucle `daemon()`

**Files:**
- Modify: `ytstock.py`

**Interfaces:**
- Produces:
  - `open_video_ids() -> set[str]` — IDs des vidéos du dossier actuellement ouvertes par un lecteur (via `lsof`).
  - `daemon()` — boucle : watcher chaque `POLL_SECS`, `sync_history` + `refill` chaque `HISTORY_SECS`.

- [ ] **Step 1: Implémenter**

```python
import time


def open_video_ids():
    try:
        out = subprocess.run(["lsof", "-F", "n", "+D", str(VIDEO_DIR)],
                             capture_output=True, text=True, timeout=30)
    except (subprocess.SubprocessError, OSError):
        return set()
    ids = set()
    for line in out.stdout.splitlines():
        if line.startswith("n"):
            vid = id_from_name(line[1:])
            if vid:
                ids.add(vid)
    return ids


def daemon():
    ensure_state_dir()
    log("daemon start")
    acc = load_watch()
    last_history = 0.0
    while True:
        try:
            watched, acc = watcher_tick(open_video_ids(), acc)
            save_watch(acc)
            did_delete = False
            for vid in watched:
                mark_watched(vid, "local")
                did_delete = True

            now = time.monotonic()
            if now - last_history >= HISTORY_SECS:
                sync_history()
                refill()
                last_history = now
            elif did_delete:
                refill()
        except Exception as e:                 # le démon ne meurt jamais
            log(f"daemon loop error: {e}")
        time.sleep(POLL_SECS)
```
